Sample Page for Working with Provengo's runCode

The runCode method of Provengo's Selenium library allows running code within the browser, and reporting back to the provengo system. This page contains a sample table that can be used to experiment with this command. Use this page to check that the row and column totals are correct, that all numbers are valid, etc.
